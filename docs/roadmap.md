# VoteChain Roadmap

This living roadmap documents VoteChain's planned milestones, priorities, and effort estimates. It is reviewed quarterly and updated as the project evolves.

## Milestone 1: MVP (Current)
Focus: core voting, wallet integration, and basic proposal listing.
Priority: High

### Features
- Proposal creation flow
  - Effort Estimate: Medium
  - Priority: P0
- Wallet integration for proposal submission and voting
  - Effort Estimate: Medium
  - Priority: P0
- Active proposals list with basic proposal details
  - Effort Estimate: Small
  - Priority: P0
- Token-weighted voting and quorum enforcement
  - Effort Estimate: Medium
  - Priority: P0
- Proposal lifecycle state management (Active → Passed/Rejected → Executed)
  - Effort Estimate: Medium
  - Priority: P0
- Basic on-chain event tracking for proposals and votes
  - Effort Estimate: Small
  - Priority: P1

## Milestone 2: v1.0 (Scaling)
Focus: gas optimizations, advanced governance modules (Quadratic Voting), and mobile responsiveness.
Priority: Medium

### Features
- Gas and transaction cost optimizations
  - Effort Estimate: Medium
  - Priority: P0
- Advanced governance module support (Quadratic Voting)
  - Effort Estimate: Large
  - Priority: P0
- Mobile-responsive UI and improved user experience for wallets
  - Effort Estimate: Medium
  - Priority: P1
- Enhanced proposal discovery and filtering
  - Effort Estimate: Small
  - Priority: P1
- Better off-chain indexer integration for proposal history
  - Effort Estimate: Medium
  - Priority: P1
- Improved analytics and voting insights
  - Effort Estimate: Medium
  - Priority: P2

## Milestone 3: v2.0 (Ecosystem)
Focus: cross-chain voting, DAO treasury integration, and delegated voting.
Priority: Strategic

### Features
- Cross-chain voting capabilities and interoperability
  - Effort Estimate: Large
  - Priority: P0
- DAO treasury integration for on-chain funding decisions
  - Effort Estimate: Large
  - Priority: P0
- Delegated voting and representative governance
  - Effort Estimate: Medium
  - Priority: P1
- Multi-token governance support and flexible weighting
  - Effort Estimate: Medium
  - Priority: P1
- Ecosystem integrations for governance tooling and dashboards
  - Effort Estimate: Large
  - Priority: P2

## Changelog/Maintenance
This roadmap is reviewed quarterly. Updates are tracked with project milestones and should reflect changes in product direction, technical priorities, and ecosystem requirements.
